Potenciação

A potenciação é uma operação matemática que representa uma multiplicação de fatores iguais de uma forma abreviada e simplificada. Em vez de escrever 2 × 2 × 2, podemos usar a potenciação.

Conceito e Notação

Quando um número (a base) é multiplicado por si mesmo várias vezes, usamos um pequeno número (o expoente) para indicar quantas vezes essa base se repete como fator.

Estrutura: baseexpoente = potência

Exemplos:

  • 2³ (lê-se "dois elevado à terceira potência" ou "dois ao cubo"):
    • Base = 2
    • Expoente = 3
    • Cálculo: 2 × 2 × 2 = 8
    • Resultado (potência) = 8
  • 5² (lê-se "cinco ao quadrado"):
    • 5 × 5 = 25
  • Exemplo prático: Uma caixa tem 3 linhas de 3 caixinhas menores. Total: 3² = 9.
  • Se cada caixinha tiver 3 cubos: 3³ = 27 cubos no total.

Casos Especiais Importantes

  • Expoente 1: Qualquer número (≠ 0) elevado a 1 é igual a ele mesmo.
    • 5¹ = 5, 100¹ = 100
  • Expoente 0: Qualquer número (≠ 0) elevado a 0 é igual a 1.
    • 7⁰ = 1, 2500⁰ = 1
    • Atenção: 0⁰ é uma indeterminação e não é considerada 1 no Nível 1 da OBMEP.

Cálculo de Potências

Para calcular uma potência, basta multiplicar a base por ela mesma o número de vezes indicado pelo expoente.

Exemplo: Calcular 3⁴:

3⁴ = 3 × 3 × 3 × 3
3 × 3 = 9
9 × 3 = 27
27 × 3 = 81
Resultado: 3⁴ = 81
